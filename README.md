# CASSINI Forecast Auth Demo

Standalone React demo for login and account-scoped access to a satellite weather warning,
evacuation planning, and resource management app.

## Run locally

```bash
npm install
npm run dev
```

The app runs on the Vite dev server, usually at `http://127.0.0.1:5173/`.
